# 4-scripting

This repository contains two simple programs:

- Bash script (`hello.sh`)
- Python script (`hello.py`)

## Bash Script

Bash is an **interpreted** scripting language. It is executed by the Bash interpreter and is **not compiled**.

### Make the script executable

```bash
chmod +x hello.sh
```

### Run the script

```bash
./hello.sh YourName
```

Example:

```bash
./hello.sh Afzal
```

---

## Python Script

Python is an **interpreted** programming language. It is executed by the Python interpreter and is **not compiled** before execution.

### Run the script

```bash
python3 hello.py YourName
```

Example:

```bash
python3 hello.py Afzal
```
